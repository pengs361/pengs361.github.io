### windows系统配置

1. 创建并配置共享目录saomiao可读写。
2. 新建saomiao账户，设置密码NXmy2021
3. 打开打印机配置页面，添加SMB注册地址，修改主机地址为主机ip,文件路径saomiao,用户id亦为saomiao,密码NXmy2021保存即可。

### 统信Uos系统配置

1. 创建并配置共享目录saomiao可读写。
2. 安装samba共享服务 `apt install samba samba-common -y && systemctl enable smbd`
3. 配置smb文件，新增字段
```
[saomiao]
   comment = saomiao
   path = /home/user/saomiao
   browsable = yes
   writable = yes
   guest ok = yes
   create mask = 0755
   force user = root
```
4. 重启SMB服务，`systemctl restart smbd`
5. 打开打印机配置页面，添加SMB注册地址，修改主机地址为主机ip,文件路径saomiao,用户id,密码均为空，保存即可。
